\subsubsection{\-Detailed \-Description}
\-The additive differential probability (\-A\-D\-P) of a modified version of the \-F-\/function of \-T\-E\-A with the shift operations removed. \-Complexity $O(n)$. \-The \-F-\/function of \-T\-E\-A with the shift operations removed is denoted by \-F' and is defined as\-: $y = F'(k_0, k_1, \delta |~ x) = (x + k_0) \oplus (x + \delta) \oplus (x + k_1)$.
